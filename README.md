# Text to ASCII Art Generator
This project provides two main to generate ascii art, either image to ascii or text to ascii.

## Usage
Image to ASCII Art Converter:
```python
python img2ascii.py house.jpg modern
```
Text to ASCII Art Converter:
```python
python text2ascii.py "llama"
```
